# AI Periodic Table for Kids

An interactive, AI-powered Periodic Table educational app designed for students (K-12).

## Features
- **Interactive Periodic Table**: Click elements to see detailed properties.
- **AI Tutor**: Ask questions about chemistry and get simple, age-appropriate answers (powered by `google/flan-t5-small`).
- **Search & Filter**: Find elements by name or symbol.

## Running Locally
This project uses `uv` for dependency management.

```bash
uv run app.py
```
